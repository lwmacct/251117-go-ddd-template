# Web Frontend

Vue 3 + Vite + Vuetify 前端应用。

## 技术栈

- **Vue 3** - Composition API + `<script setup>`
- **Vite** - 构建工具
- **Vuetify 3** - UI 组件库
- **VueUse** - 工具函数库
- **Pinia** - 状态管理
- **TypeScript** - 类型安全

## 开发

```sh
pnpm install
pnpm dev
```

## 构建

```sh
pnpm build
pnpm preview
```
